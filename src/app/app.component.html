Pxnutri
Sistema para nutricionistas
O PxNutri é um software de automação desenvolvido por mim e minha esposa Pâmela com muito carinho, o criamos para proporcionarmos praticidade, agilidade e organização. Nós facilitamos a comunicação entre nutricionista e cliente. Através desta ferramenta, o nutricionista consegue acompanhar com mais facilidade o tratamento do paciente. Ele tem vantagens e recursos exclusivos para melhor experiência tecnológica para Nutricionistas e Pacientes. Foi feito utilizando Python e Angular
Diagrama
Sistema de diagrama 'UML'
Esboço de um diagrama com scroll de tela e conexão de tabelas do modelo entidade-relacionamento no html utilizando apenas angular e ngx-leader-line para tal finalidade. Potencial programa para utilização de modelagem interna dos sistemas criados pela PxTech.
Pontuei
Sistema de pontuação (placar)
O Pontuei é um sisteminha pessoal que utiliza angular e django para registrar o placar de forma virtual de uma partida de jogos de baralho, tabuleiro e outros.
